# My Portfolio

This is a React project created using Vite and styled with Tailwind CSS. The project serves as a personal portfolio to showcase projects and skills.

## Getting Started

To get started with this project, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone <repository-url>
   cd my-portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Open your browser:**
   Navigate to `http://localhost:3000` to view your application.

## Project Structure

The project has the following structure:

```
my-portfolio
├── index.html          # Main HTML document
├── package.json        # NPM configuration file
├── tsconfig.json       # TypeScript configuration file
├── vite.config.ts      # Vite configuration file
├── tailwind.config.cjs  # Tailwind CSS configuration file
├── postcss.config.cjs  # PostCSS configuration file
├── .gitignore          # Git ignore file
├── README.md           # Project documentation
└── src
    ├── main.tsx       # Entry point of the React application
    ├── App.tsx        # Main App component
    ├── index.css      # Global CSS styles
    ├── components
    │   └── Portfolio.tsx # Portfolio component
    └── types
        └── index.d.ts  # TypeScript types and interfaces
```
